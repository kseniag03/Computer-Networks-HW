Структура проекта

/src/main/java/org/example -- здесь все написанные классы

Main -- запуск; ввод таймаута, инициализация основных сущностей (интерфейса, интерпретатора)
CommandInterpreter -- класс для работы с командами (там же есть список команд)

/Common -- общие классы для проекта
Constants -- основные константы для использования

/Utils -- используемые утилиты
ShutdownHandler -- закрытие ресурсов в случае досрочного завершения программы
ArpTargetedRequestsSender -- класс для выделения логики запроса
ArpCapture -- класс для захвата и вывода пакетов
DuplicateIpDetector -- класс для выяснения, есть ли в сети устройство с аналогичным IP адресом
MacDiscover -- класс для получения и вывода MAC адресов
StatisticsCollector -- класс для сбора и вывода статистики

Структура проекта

/src/main/java/org/example — здесь все написанные классы, структура аналогична ДЗ №2
классы по Arp и Dns протоколам разделены на папки Utils/Arp и Utils/Dns

Main — запуск; инициализация основных сущностей (интерфейса, интерпретатора)
CommandInterpreter — класс для работы с командами (там же есть список команд) — модифицирован под ДЗ №3

/Common — общие классы для проекта
Constants — основные константы для использования

/Utils/Dns — используемые утилиты для работы с протоколом Dns
ShutdownHandlerDns — закрытие ресурсов в случае досрочного завершения программы
DnsCapture — захватить DNS пакеты и вывести их в консоль
MxIpDetector — найти IP почтового сервиса из MX-записи
RootServerHandler — отправить запрос на корневой сервер для резолва домена
